Linux Commands Cheat Sheet

A comprehensive reference guide for common Linux commands and their usage.

Hardware Information
lscpu
See CPU information.
lsblk
See information about block devices.
lspci -tv
Show PCI devices in a tree-like diagram.
lsusb -tv
Display USB devices in a tree-like diagram.
lshw
List hardware configuration information.
cat /proc/cpuinfo
Show detailed CPU information.
cat /proc/meminfo
View detailed system memory information.
cat /proc/mounts
See mounted file systems.
free -h
Display free and used memory.
sudo dmidecode
Show hardware information from the BIOS.
hdparm -i /dev/[device]
Display disk data information.
hdparm -tT /dev/[device]
Conduct a read speed test on the device/disk.
badblocks -s /dev/[device]
Test for unreadable blocks on the device/disk.
fsck /dev/[device]
Run a disk check on an unmounted disk or partition.
Searching
find [path] -name [pattern]
Find files and directories matching the pattern.
find [path] -size [+100M]
See files and directories larger than a specified size.
grep [pattern] [file]
Search for a specific pattern in a file.
grep -r [pattern] [dir]
Recursively search for a pattern in a directory.
locate [name]
Locate all files and directories related to a particular name.
which [command]
Search the command path in the $PATH variable.
whereis [command]
Find the source, binary, and manual page for a command.
awk '[pattern] {print $0}' [file]
Print all lines matching a pattern in a file.
sed 's/[old]/[new]/' [file]
Find and replace text in a specified file.
Files
mkdir [dir]
Create a new directory.
rm [file]
Remove a file.
rm -r [dir]
Remove a directory recursively.
rm -rf [dir]
Recursively remove a directory without confirmation.
cp [source] [dest]
Copy the contents of one file to another.
cp -r [source_dir] [dest_dir]
Recursively copy a directory.
mv [source] [dest]
Move or rename files or directories.
ln -s [path]/[file] [link]
Create a symbolic link to a file.
touch [file]
Create a new file.
cat [file]
Show the contents of a file.
cat [source] >> [dest]
Append file contents to another file.
head [file]
Show the first ten lines of a file.
tail [file]
Show the last ten lines of a file.
more [file]
Display contents of a file page by page.
less [file]
Show the contents of a file with navigation.
nano [file]
Open or create a file using nano.
vi [file] / vim [file]
Open or create a file using Vi/Vim.
gpg -c [file]
Encrypt a file.
gpg [file].gpg
Decrypt an encrypted .gpg file.
wc -w [file]
Show the number of words, lines, and bytes in a file.
ls | xargs wc
List line/word/char counts for each file in a directory.
cut -d [delimiter] [file]
Cut a section of a file and print the result.
[data] | cut -d [delimiter]
Cut a section of piped data and print the result.
shred -u [file]
Overwrite a file to prevent recovery, then delete it.
diff [file1] [file2]
Compare two files and display differences.
source [file]
Read and execute the file content in the current shell.
[command] | tee [file] >/dev/null
Store command output in a file and skip terminal output.
